## Canker Core

Canker Core is web app for tracking and managing your mouth sores. It is build with Next.js, TypeScript, and Supabase.

## Features

- [x] Authentication
- [x] Add new sores
- [x] Edit sores
- [x] Delete sores
- [x] View all sores

## Tech Stack

- Next.js
- TypeScript
- Supabase
- Tailwind CSS
- ShadCN
